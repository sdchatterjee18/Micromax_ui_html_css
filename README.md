# Micromax Website UI (HTML & CSS)

A **Micromax-inspired website user interface** built using **pure HTML and CSS**, without any JavaScript.  
This project is created for **learning and practice purposes** to understand real-world website layouts and styling.

---

## 🚀 Project Overview

This project focuses on building a complete website UI using only frontend fundamentals.  
It helped me practice layout structuring, styling, and hover-based interactions commonly used in modern websites.

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Flexbox  
- Font Awesome Icons  

---

## ✨ Features

- Fixed navigation bar  
- Hero banner with text overlay  
- Product cards layout  
- Hover effects & smooth transitions  
- Clean and modern UI design  
- Fully built **without JavaScript**

---

## 📚 What I Learned

As a beginner, this project helped me learn and practice:

- Website layout using Flexbox  
- Positioning elements (relative & absolute)  
- Hover effects using `:hover`  
- CSS transitions and transforms  
- Creating reusable sections (cards, banners, footer)  
- Structuring HTML for real-world websites
